🏗️ 一、基础开发环境

1.1 MongoDB 本地开发配置

类比理解：

  • MongoDB ≈ 数据库服务器（相当于MySQL）
  • mongosh ≈ 数据库客户端（相当于MySQL Workbench）
  • 浏览器 ≈ Web服务器客户端

1.2 Windows管理员终端快捷启动

  1. 按 Win + R 打开运行对话框
  2. 输入 cmd
  3. 按 Ctrl + Shift + Enter 以管理员身份运行

1.3 NVM (Node版本管理工具)

文档未详述此点，但基于我所掌握的知识：nvm是管理多个Node.js版本的工具

核心功能：

  • 多版本管理：一台设备安装多个Node.js版本
  • 版本切换：按项目需求切换特定版本
  • 兼容性解决：处理不同项目的版本依赖

安装注意事项（来自链接内容）：

  • 安装脚本会自动配置环境变量
  • 如遇"nvm: command not found"，尝试：
    • 关闭终端重新打开
    • 执行 source ~/.bashrc 或相应shell配置文件
    • macOS用户需注意默认shell为zsh，需检查.zshrc文件
  • 可手动将nvm配置代码添加到shell配置文件中

⚡ 四、现代前端框架

4.1 Vue.js 核心概念

v-if vs v-show：

  • v-if：真正的条件渲染，切换时销毁/重建DOM元素，有更高的切换开销
  • v-show：始终渲染，只是切换CSS的display属性，有更高的初始渲染开销
  • 使用场景：频繁切换用v-show，运行时条件很少改变用v-if

📊 五、开发知识体系结构

现代Web应用开发层次

蓝桥杯考点总结

以下是您文档中提到的核心考点分类，适合复习参考

HTML+CSS

  • 基础标签：HTML结构、行内/块级标签、表单
  • HTML5：语义化标签、多媒体、本地存储
  • CSS：选择器、盒子模型、浮动定位
  • CSS3：文本溢出、边框阴影、渐变、过渡动画
  • 布局：Flex、Grid、媒体查询

JavaScript

  • 基础：变量、数组、运算符、分支循环
  • DOM/BOM：window、document、location
  • 内置对象：Math、Date、Array、String
  • 事件：鼠标键盘事件、event对象
  • AJAX、正则表达式

现代技术栈

  • ES6+：let/const、class、Promise、async/await
  • Vue生态：指令、组件、生命周期、Vue Router
  • UI库：Element Plus组件使用
  • 图表：ECharts基础、图表绘制
  • 状态管理：Pinia (State、Getters、Actions)
  • Node.js：服务器创建、内置模块

🎯 六、实用设计指南

毛玻璃+透明悬浮卡片风格

  1. 毛玻璃背景：backdrop-filter: blur() + 半透明背景
  2. 卡片式布局：独立悬浮卡片，错落有致
  3. Hover动效：轻微上浮、发光边框
  4. 色彩方案：低饱和度 + 紫色调渐变背景
  5. 字体图标：现代无衬线字体 + 线性图标风格

推荐资源

  • 字体：San Francisco、Inter
  • 图标库：Feather Icons、Lucide
  • 设计工具：Figma
  • 部署平台：Netlify、Vercel

💡 使用建议

  1. 学习路径：按照"基础 → 框架 → 项目 → 优化"顺序学习
  2. 实践方法：每学一个知识点，立即手写代码验证
  3. 复习策略：定期回顾此文档，重点关注标记的"高频考点"
  4. 项目驱动：尝试用所学技术实现一个小型完整项目
  5. 查漏补缺：根据蓝桥杯考点列表，逐个检查掌握程度
