watch是用于监听响应式数据变化的函数

  1. 监听单个响应式数据
    watch(()=>props.worker.name,(newVal,oldVal)=>{})
  2. 监听多个响应式数据
    watch([()=>props.worker.name,()=>props.worker.phone],
    ([newName,newPhone],[oldName,oldPhone])=>{})
  3. 直接监听ref
    import{ref,watch}from ‘vue’
    Const count=ref(0)
    watch(count,function())
  4. 监听reactive对象的属性
    import {reactive,watch}from ‘vue’
    Const state=reactive({
    name:’’,age:0})
    watch(()=>state.name,(newVal,oldVal)=>{})
  5. 深度监听

  1. 什么是 v-model？
    v-model 是 Vue 的指令，用于在表单元素和数据之间创建双向绑定

  2. 在这个场景中的具体作用：
    v-model=”item.checked“ 将复选框的选中状态与员工对象的 checked 属性绑定

双向绑定意味着：

  • 数据 → 视图：当 item.checked 值为 true/false 时，复选框会自动选中/取消
  • 视图 → 数据：当用户点击复选框时，item.checked 的值会自动更新
