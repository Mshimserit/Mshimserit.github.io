两端对齐分布

order,align-self

  1. order: 1;
    控制元素的排列顺序。默认情况下，所有 Flex/Grid 子项的 order值都为 0。数值越大，排列顺序越靠后。
    将此值设为 1，意味着这个元素会排在所有 order值小于 1（即默认值为 0）的其他兄弟元素之后。
  2. align-self: flex-end;
    在 Flex 容器中，它控制该元素在交叉轴（默认是垂直方向）上的对齐方式，并单独覆盖容器为所有子项设置的 align-items规则。
    设为 flex-end会使这个元素紧贴容器在交叉轴的末端对齐。
    如果 Flex 容器是水平的 (flex-direction: row)，flex-end表示底部对齐。
    如果 Flex 容器是垂直的 (flex-direction: column)，flex-end表示右对齐（或左对齐，取决于书写方向）。
  3. opacity:0.5;
    将元素的透明度设为 50%。这意味着元素本身（包括其所有内容：文字、背景、边框、子元素等）都会变成半透明。只想让元素的背景颜色变透明:rgba()

grid布局

grid-area:2/2;
grid-area: / 是 grid-area的一种简写形式。

grid-area:grid-row-start / grid-column-start / grid-row-end / grid-column-end
grid-area: 1 / 2 / 3 / 4;
grid-row-start: 1; - 从第 1 条行网格线开始。
grid-column-start: 2; - 从第 2 条列网格线开始。
grid-row-end: 3; - 到第 3 条行网格线结束。这意味着该网格项在垂直方向上跨越了 2 行（第1行和第2行）。
grid-column-end: 4; - 到第 4 条列网格线结束。这意味着该网格项在水平方向上跨越了 2 列（第2列和第3列）。

blur()

需要对包含人物和人物身后枫叶林的 img 标签设置清晰效果，对人物与枫叶林之外的 img 标签应保持模糊效果
document.querySelector(‘.img’).style.filter=’blur(0px)’

1.元素的filter样式属性用于指定一个或多个滤镜效果，从而改变元素的外观。
blur()：模糊元素。参数值表示模糊程度，单位为像素。
brightness()：调整元素的亮度。参数值表示亮度的百分比。
contrast()：调整元素的对比度。参数值表示对比度的百分比。
grayscale()：将元素转换为灰度图像。参数值表示灰度的百分比。
hue-rotate()：调整元素的色相。参数值表示旋转角度，单位为度数。
invert()：反转元素的颜色。参数值表示反转的百分比。
opacity()：调整元素的不透明度。参数值表示不透明度的百分比。
saturate()：调整元素的饱和度。参数值表示饱和度的百分比。
sepia()：将元素转换为褐色图像。参数值表示褐色的百分比。
